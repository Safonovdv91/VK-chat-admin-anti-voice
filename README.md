# Бот удаляющий голосовые сообщения и видео кружки определенного времени

При получении голосового сообщение в групповом чате, его репоста, получении видеокружка бот удаляет его через настроенное время, после чего удаляет настроечное сообщение.

1. Создать .env в корневом каталоге

```.env
VK_TOKEN=YOUR-VK-TOKEN
MAX_VOICE_DURATION = 20 # Масимальная разрешенная длительность голосового сообщения
MAX_VIDEO_DURATION = 20 # Масимальная разрешенная длительность видеокружка
```

1. Запустить бот

```
python main.py
```

# Пайплайн использования

1. Создаете группу.
2. Получаете в настройка vk_api, разрешаете. long_polling
3. Добавить группу в чат(через функционал ВК-группы)
4. Дать права администратора для бота(без этих прав нет возможности читать и удалять сообщения)
